La méthode "inline"

Consiste à rajouter un attribut dans la balise cible

Titre avec attribut style

        <h2 style="color": red">

La balise <style>

Ouverture d'une balise style en début de body permettant l'ajout de propriétés CSS

        <style>
            h2 {
                    color: orange;
                }
        </style>
        <h2>Titre en orange</h2>

La feuille de style

Création d'une feuille de style contenant l'ensemble des propriétés CSS.

Feuille de style liée dans la balise <head> du document.

        <link rel="stylesheet" href="style.css type="text/css">
