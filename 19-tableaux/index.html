Les tableaux

Structure d'un tableau

        // Balise parente 
        <table>
            // Header du tableau
            <thead>
                // Ligne du tableau
                <tr>
                    // Titre de la colonne
                    <th>
                    </th>
                </tr>
            </thead>
            // Body du tableau
            <tbody>
                <tr>
                    //Contenu du tableau
                    <td>
                    </td>
                <tr>
            </tbody>
            // Footer du tableau
            <tfoot>
                <tr>
                    //Contenu du tableau
                    <td>
                    </td>
                <tr>
            </tfoot>
        </table>

Attributs des balises <td>

  • colspan : étend la cellule sur le nombre de colonne spécifié
  • rowspan : étend la cellule sur le nombre de ligne spécifié

Démo :

Les éléments sont représentés sous la forme colonne:ligne

Titre colonne 0 Titre colonne 1 Titre colonne 2 Titre colonne 3
0:0 1:0 2:0 3:0
0:1 1:1 2:1 3:1
0:2 1:2 2:2 3:2
