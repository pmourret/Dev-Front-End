Audios

Balise : <audio>

Attributs

  • controls : affiche le lecteur
  • autoplay : lecture automatique
  • loop : lecture en boucle

Videos

Balise : <video>

Attributs

  • poster : image de couverture
  • controls : affiche les controles
  • autoplay : lecture automatique
  • loop : lecture en boucle
  • muted : son muté

Images

Balise : <image>

Attributs

  • src : source du média
  • alt : texte alternatif

Balise : <figure>

Permet d'afficher différentes images en fonctions de la taille de l'écran

Structure

        <figure>
            <picture>
                <source>
                <source>
                <img>
                <figcaption>
            <picture>
        <figure>

Attributs

  • media : choix de la taille minimale d'écran
  • srcset : média source
  • type : type du média
