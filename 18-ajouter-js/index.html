La balise <script>

  • Directement dans la balise <head>

                    <script>
                    alert('Je suis une alerte depuis le head') ;
                </script>

  • Directement dans la balise <body>

                    <script>
                    alert('Je suis une alerte depuis le body') ;
                </script>

Depuis un fichier externe

Création d'une feuille de script nécessaire à l'application.

Ouverture d'une balise <script> avant la fermeture du body

    <script src="app.js"></script>

Attributs de <script>

  • defer : précise que le script doit être utilisé avant le chargement de la page
  • type : précise le type de script utilisé
  • integrity : vérifie l'intégrité d'un script pour un script distant
