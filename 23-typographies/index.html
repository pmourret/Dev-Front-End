Les typographies

Obtenir des typographies peut se réaliser sur Google Fonts.

Afin d'intégrer la police choisie deux options

  • Dans une balise link située dans le <head>
  • Par un import dans une feuille de style CSS
        
// Méthode n°1

        
// Lien vers la police

        <head>
            < rel="preconnect" href="https://fonts.googleapis.com">
            < rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
            < href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@1,100&display=swap" rel="stylesheet">
        </head>

        
// Définition de règle CSS => Appliquer la police à tous les éléments du <body>

        <style>
            body{
                font-family: 'Roboto', sans-serif;
            }
        </style>
    
        
// Méthode n°2

        <style>
            
// Import de la police

            @import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@1,100&display=swap');

            
// Définition de règle CSS => Appliquer la police à tous les éléments du <body>

            body{
                font-family: 'Roboto', sans-serif;
            }

        </style>
