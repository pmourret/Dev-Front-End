La balise META OG

La balise META OG sert à "booster" les pages WEB au niveau du SEO

Exemple

Exemple pour une page web contenant un article

        Dans la balise head

        Pour le titre de l'article
        <meta property="og:title" content="Titre de l'article" >
        Pour afficher un lien
        <meta property="og:url" content="Lien vers une page" >
        Pour la langue du site web
        <meta property="og:local" content="fr" >
        Pour afficher une image
        <meta property="og:image" content="LIEN VERS IMAGE" >
